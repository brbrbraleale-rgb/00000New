# Проект: Учет товаров и категорий

Учебный проект на Python для управления каталогом товаров. Реализованы классы для представления товаров и их категорий с автоматическим подсчетом данных.

## Функционал
- **Product**: Класс для хранения данных о товаре (название, описание, цена, количество).
- **Category**: Класс для группировки товаров. 
  - Ведет автоматический учет общего количества созданных категорий.
  - Ведет учет общего количества товаров.
  - Созданы специальные классы-наследники от базового класса Product, и новые методы проверки и сложения,что позволяет более точно описывать товары в магазине и
Безопасно добавлять продукты в категории
